Latitude with Attitude

Does it really get hotter when we get closer to the equator?

We have created a visualization dashboard website using visualizations plotting weather data, in order to analyze how weather changes as you get closer to the equator. To accomplish this analysis, data was pulled from the OpenWeatherMap API to create a dataset for over 500 cities. A variety of different factors were compared to Latitude. For this website we have demostrated 4 individual pages for 4 different plots:
1) City Latitude vs Humidity
2) City Latitude vs Cloudiness
3) City Latitude vs Wind Speed
4) City Latitude vs Max Temperature

Each plot has its own page explaining the date. There is also a comparison page where we can see a comparison of all of the plots, and another page where we can view the data used to build them.
